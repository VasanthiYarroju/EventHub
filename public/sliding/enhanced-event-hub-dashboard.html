Help Center

Quick Start Guide

  • Creating an Event: Click the "Add New Event" button and fill out the form.
  • Managing Events: View all your events in the table. Click on an event to see details.
  • Tracking Progress: Update event status and add tasks to the event checklist.
  • Searching: Use the search box to find specific events.
  • Filtering: Use the dropdown menus to filter events by status or type.

Frequently Asked Questions

How do I share my event with others?

Open the event details and click the "Share" button. You can then invite team members via email or generate a shareable link.

Can I export my events to a calendar?

Yes! Click the "Export" button and select "Calendar (iCal)" to download a file compatible with most calendar applications.

How do I set reminders for my events?

Open the event details, go to the "Reminders" tab, and set your preferred notification schedule.
